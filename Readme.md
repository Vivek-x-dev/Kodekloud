##  Repository Structure

 **Repository Structure**

- `README.md`
- `logs/` — Daily write-ups (day-01.md ... day-100.md)
- `scripts/` — Bash/Python scripts I write
- `configs/` — Dockerfiles, K8s manifests, Terraform HCL, Ansible playbooks


## My Daily Approach

1. Pick the day's challenge.
2. Solve it hands-on (terminal, cloud, or local lab).
3. Document the solution, commands used, and key takeaways in `logs/day-XX.md`.
4. Commit and push.
